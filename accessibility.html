Accessibility Statement

Last updated: July 2026

Our commitment

Neorons is committed to making this website accessible to the widest possible audience, including people with disabilities. Our mission is inclusion — our website should reflect that.

Conformance

This website aims to conform to the Web Content Accessibility Guidelines (WCAG) 2.1 at Level AA. We test with keyboard navigation, screen readers (VoiceOver on macOS/iOS), and automated contrast checks.

Accessibility features

  • Keyboard navigation: All interactive elements are reachable and operable via keyboard. The event modal includes a focus trap and Escape-to-close.
  • Screen reader support: ARIA labels, roles, and live regions provide context. Filter changes are announced. Modal navigation is labelled.
  • Reduced motion: All animations and transitions respect the prefers-reduced-motion system setting. With motion reduced, content is visible immediately without animation.
  • Skip link: A "Skip to main content" link is the first focusable element on every page.
  • Contrast: Text colours meet WCAG AA contrast ratios against their backgrounds.
  • Alt text: All meaningful images have descriptive alternative text.
  • No autoplay: No audio or video plays automatically.

Known limitations

The event timeline component requires horizontal scrolling which may be less accessible to some users. The district grid is provided as a fully accessible alternative.

Reporting barriers

If you experience any difficulty accessing this website, please contact us at hello@neorons.org. We take every report seriously and will work to resolve barriers promptly.
